.. _Select feature subset from classification sample:

************************************************
Select feature subset from classification sample
************************************************

Select a feature subset from given sample.

**Parameters**


:guilabel:`Sample` [file]
    Sample (*.pkl) file.


:guilabel:`Selected features` [string]
    List of selected features given by name (e.g. ´Feature 1´) or position (integer >=1).

**Outputs**


:guilabel:`Output sample` [fileDestination]
    Output sample *.pkl file.
